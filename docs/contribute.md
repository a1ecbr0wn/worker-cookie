## Contribute

Contributions are welcome. Areas where help would be appreciated:

- **New themes** — if you design a new banner theme, a PR adding it to `assets/themes.css` and documenting it here would be great.
- **New locales** — the example config ships with `en_GB`, `fr_FR`, and `de_DE`. Contributions of additional locale examples in the `config/` directory are welcome.

Please raise a PR on [GitHub](https://github.com/a1ecbr0wn/worker-cookie).
